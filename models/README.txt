The GitHub source repository does not store .pth model files directly.

Download the single-file installer from Latest Release:

ImageSuperResolutionTool-EN-v1.1.3-Setup.exe

The installer includes four super-resolution models:

- 003_ATD_SRx4_finetune.pth
- Real_HAT_GAN_sharper.pth
- RealESRGAN_x4plus.pth
- RealESRGAN_x4plus_anime_6B.pth

The app uses ATD Official Super-Resolution Model by default. You can also switch to HAT High-Quality Super-Resolution Model, Real-ESRGAN General Photo Model, or Real-ESRGAN Anime and Illustration Model in the app.
