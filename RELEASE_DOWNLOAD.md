# Download Guide

For regular users, download the single installer from Latest Release:

`ImageSuperResolutionTool-EN-v1.1.2-Setup.exe`

Latest Release:
https://github.com/qwertasdfg77/image-super-resolution-tool-en/releases/latest

## Usage

1. Download `ImageSuperResolutionTool-EN-v1.1.2-Setup.exe`.
2. Run it.
3. Choose an install location.
4. Open the app from the desktop shortcut after installation.
5. On first use, click `Install/Check Runtime` in the app.

## Current Version

`v1.1.2` is the single-file installer release:

- Includes four super-resolution models.
- Supports custom install locations.
- Creates a desktop shortcut after installation.
- Does not pre-bundle `.venv`; the app installs the local Python runtime, CUDA PyTorch, and other dependencies through `Install/Check Runtime`.
- Does not include sample photos.
