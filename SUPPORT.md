# Support

## Usage Issues

For regular usage questions, open an Issue:

https://github.com/qwertasdfg77/image-super-resolution-tool-en/issues

When reporting a problem, include:

- Tool version, for example `v1.1.2`
- Windows version
- NVIDIA GPU model and VRAM
- Install location
- Selected model: ATD Official Super-Resolution Model, HAT High-Quality Super-Resolution Model, Real-ESRGAN General Photo Model, or Real-ESRGAN Anime and Illustration Model
- Status text or screenshot from the app window
- Input image format and dimensions

## Support Scope

Currently focused on:

- Windows 10 / Windows 11
- NVIDIA CUDA GPUs
- Recommended RTX 4060 8GB or higher
- Image formats: JPG, PNG, WEBP, BMP, TIF/TIFF

Not currently a primary focus:

- macOS / Linux GUI packaging
- AMD / Intel GPU acceleration
- Real-time in-game DLSS
